プライバシーポリシー

このアプリは、以下に記載された情報を収集する場合があります。

収集される情報

収集された情報は、以下の場合を除いて第三者と共有されることはありません。

  • お客様から明示的な同意を得た場合

収集された情報の保管

収集された情報は、適切な保管措置を講じて安全に保管されます。
